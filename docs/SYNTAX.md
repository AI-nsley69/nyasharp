```
// - comments

o/ - assignment
>w< - constant declaration
>.< - variable declaration

\o/ - equal to
_o/ - equal to or less than
\o_ - equal to or greater than
/o/ - less than
\o\ - greater than
_o_ - not equal to

+.+ - addition operator
-.- - subtraction operator
+.* - multiplication operator
-.* - division operator

^u^ - if statement
^o^ - for loop
^w^ - while loop
:D - function declaration
c: - return

:> // this is a code block <:
```
